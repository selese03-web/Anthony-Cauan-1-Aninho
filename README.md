## 🔧 Instalação e Uso

### Como Usar

1. **Clone o repositório:**
```bash
git clone https://github.com/selese03-web/Anthony-Cauan-1-Aninho.git
cd Anthony-Cauan-1-Aninho
```

2. **Abra o arquivo HTML:**
   - Duplo clique em `index.html`, ou
   - Use um servidor local (recomendado):
   ```bash
   # Com Python 3
   python -m http.server 8000
   
   # Com Node.js (http-server)
   npx http-server
   ```

3. **Acesse no navegador:**
   - Localmente: `http://localhost:8000`

## 📝 Personalizações

### Alterar Data e Horário
No arquivo `index.html`:
```html
<p>20/09/2026</p>  <!-- Data -->
<p>14:00</p>       <!-- Horário -->
```

### Alterar Número de WhatsApp
No arquivo `index.html`:
```html
https://wa.me/5583998174426?text=...
```

### Ajustar Cores
No arquivo `style.css`:
```css
:root {
    --cor-primaria: #A9D8FF;
    --cor-secundaria: #FFE89A;
    --cor-terciaria: #CDECCB;
    --cor-branca: #FFFFFF;
    --cor-destaque: #E9C46A;
}
```
